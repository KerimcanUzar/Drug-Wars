# Drug Wars

### DESCRIPTION
	A simple TBS game that is inspired by Drug Wars by John E. Dell (1984). Made with Godot Engine (v3.4) by using GDScript.

### KNOWN PROBLEMS
	Haven't tested for possible balance issues.

### GAME LENGTH
	5-15 mins per game.
